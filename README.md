# Global Commerce Council

Official website for globalcommercecouncil.com - A mobile-first, SEO-optimized site built with Hugo, Bootstrap 5, and deployed on Cloudflare Pages.

## Features

- Mobile-first responsive design
- SEO optimized with structured data
- Fast UX with sticky header and search
- Category pages (Ceramics, Gypsum, Petroleum, Other)
- Product listings with schema.org markup
- HS Code and Supplier directories
- Serverless contact forms
- Professional WorldEconomicForum-inspired design

## Development

```bash
# Install Hugo Extended (v0.121.1 or later)
# Build the site
hugo

# Run development server
hugo server -D
```

## Deployment

Automated deployment to Cloudflare Pages via GitHub Actions.
