## استعلام کد تعرفه گمرکی HS Code

کد HS (سیستم هماهنگ شده) یک سیستم شماره‌گذاری استاندارد بین‌المللی برای طبقه‌بندی کالاها در تجارت جهانی است. در این صفحه می‌توانید کدهای HS محصولات مختلف را جستجو کنید.

### کدهای HS پرکاربرد

| محصول | کد HS |
|--------|-------|
| قیر نفتی | ۲۷۱۳۲۰ |
| گازوئیل | ۲۷۱۰۱۹ |
| نفت کوره | ۲۷۱۰۲۰ |
| نفتا | ۲۷۱۰۱۲ |
| حلال ۴۰۲ | ۲۷۱۰۱۲ |
| سود سوزآور | ۲۸۱۵۱۲ |
| گوگرد | ۲۵۰۳۰۰ |
| نمک صنعتی | ۲۵۰۱۰۰ |
| کاشی سرامیک | ۶۹۰۷۱۰ |
| تخته گچی | ۶۸۰۹۱۱ |
| هیپوکلریت کلسیم | ۲۸۲۸۱۰ |
| کلسیم کلرید | ۲۸۲۷۲۰ |

| لاستیک پلی‌بوتادین | ۴۰۰۲۲۰ |

### چگونه از کد HS استفاده کنیم؟

۱. کدهای ۲ رقمی اول: فصل کالا را مشخص می‌کند
۲. کدهای ۴ رقمی: گروه یا ردیف کالا را مشخص می‌کند
۳. کدهای ۶ رقمی: زیرمجموعه کالا را مشخص می‌کند
۴. ارقام ۸ تا ۱۰ رقمی: کدهای ملی و منطقه‌ای

برای استعلام دقیق کد HS محصول خود، لطفاً با کارشناسان ما تماس بگیرید.
